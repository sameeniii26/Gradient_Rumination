# Gradient_Rumination
# Rumination Functional Gradient Analysis

This repository contains code used in the study:

Functional connectome gradient profile of rumination state in major depressive disorder.

## Environment
MATLAB R2022a  
BrainSpace toolbox
R 4.4.2
Python 3.13.5

## Steps
1. Preprocess fMRI data using DPABI
2. Cal Gradient using BrainSpace based on Matlab
3. Cal Gradient Metrics
